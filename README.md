# Projeto: Técnicas de Desenvolvimento de Algoritmos

Este repositório consolida os projetos e exercícios desenvolvidos para a disciplina de **Técnicas de Desenvolvimento de Algoritmos**. O principal objetivo foi aplicar diferentes estratégias de projeto e análise de algoritmos para resolver problemas computacionais específicos, com foco na eficiência e complexidade.

## Tópicos Abordados

*   **Análise de Complexidade:** Estudo de notações de Big O, Omega e Theta.
*   **Estruturas de Dados:** Implementação e uso de listas, pilhas, filas, árvores e grafos.
*   **Algoritmos de Ordenação:** Implementações de Bubble Sort, Insertion Sort, Merge Sort, Quick Sort, etc.
*   **Busca e Travessia:** Algoritmos de busca binária, BFS (Busca em Largura) e DFS (Busca em Profundidade).
*   **Técnicas de Projeto:** Aplicação de Divisão e Conquista, Programação Dinâmica e algoritmos Gulosos (Greedy Algorithms).
